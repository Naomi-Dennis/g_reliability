## Testing
Testing framework is `ExUnit`. Use `mix test` to run tests.

## Start Local Server:

	- `mix deps.get
	- mix ecto.migrate
	- (Optional - Seed database) mix run priv/repo/seeds.exs
	- mix phx.server`
	- View data in browser http://localhost:4000
